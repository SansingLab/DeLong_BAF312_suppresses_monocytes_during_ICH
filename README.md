# DeLong\_BAF312\_suppresses\_monocytes\_during\_ICH
This repository contains the scripts to produce the figures in DeLong, et al. "Intracerebral hemorrhage induces monocyte TNF signaling in patients that is suppressed by BAF312."

Scripts are presented in their recommended order of execution. RData files are not included, but the RData files can be produced by executing the scripts in order. Raw and processed RNA-sequencing data are available in the PubMed Gene Expression Omnibus (GEO) database under the accession number GSE313043
